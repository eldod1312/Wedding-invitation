# Fady & Nardeen Wedding Invitation

A production-ready wedding invitation web app built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- Luxury white and gold responsive design
- Animated landing screen
- Double-door opening invitation reveal
- Main invitation, wedding date, countdown, and event details
- Google Maps button for Sunny Days
- Thank-you section
- Smooth scroll-based reveal animations

## Getting Started

```bash
npm install
npm run dev
```

The app will run on the local Vite URL shown in your terminal, usually `http://localhost:5173`.

## Build

```bash
npm run build
```

The production files will be generated in `dist/`.

## Upload To GitHub

Upload the project files to GitHub, but do not upload `node_modules`, `dist`, `work`, or `outputs`. They are ignored by `.gitignore`.

## Customize

Invitation details live in `src/data/invitation.js`. Update the wedding date, time, venue, or map link there.
